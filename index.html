Java

1.История языка.

Java – широко используемый язык программирования для написания интернет-приложений. Разработка Java началась в 1990 году, первая официальная версия — Java 1.0, — была выпущена только 21 января 1996 года. Java – это много платформенный язык.

2.Где используесться Java.

На языке Java написаны такие программы, игры и сайты как:Minecraft,Netflix,Spotify.

3.Почему Java.

Java надежный, удобный и простой язык. Язык Java создавался с возможностью обнаружения возможных ошибок, а также исключению ситуаций, которые могут привести к ошибкам. Java обеспечивает несколько уровней зашиты от вредоносного ПО. Язык Java простой в изучении и эффективный в употреблении профессиональными программистами.

Python

1.История Python.

Python разработан Гвидо Ван Россумом (Guido Van Rossum), программистом из Нидерландов. Он начал работу над языком в 1989 году , но первая версия python была опубликованна 20 февраля 1991 года. Изначально язык был полностью любительским проектом.

2.Где используется Python.

Python использовался для создания Battlefield, WoT, Sims. Python вполне применим для работы над играми, но чаще используется в других сферах.

3.Почему Python

Python прост, логичен, имеет понятный синтаксис, поэтому подходит даже для новичков. Кроссплатформенность. Программы, написанные на Python, могут запускаться и функционировать на всех типах операционных систем. Скорость разработки. Чтобы написать программу на Python нужно значительно меньше кода, чем при разработке, например — на Java.

Java Script

1.История Js.

Js был разработан бренданом эйхом.Анонсирован этот язык был 18 июля 1996 года. Первым браузером, поддерживающим эту реализацию, был Internet Explorer 3.0.На JavaScript создают «внешний интерфейс» вебприложений.  

2.Где используется Js.

JavaScript используется в коде для популярных сервисов вроде Google Maps, Google Docs, Netflix и eBay.

3.Почему Js.

Высокая скорость работы и производительность. Этот язык позволяет частично обрабатывать веб страницы на стороне пользователя. Это позволяет тратить меньше времени на открытие, а также снижает нагрузку на серверы. Относительная простота. Написание программ занимает обычно меньше времени. При этом, объем кода также обычно меньше, в сравнении со многими другими языками. Широкие возможности для веб-страниц. С помощью JS вы можете оживить любую страницу в сети Интернет. Помимо этого, JS значительно повышает уровень юзабилити приложений и сайтов.

С++

1.История С++.

Создателем языка C++ является Бьерн Страуструп В 1985 году вышло первое издание «Языка программирования C++», обеспечивающее первое описание этого языка, это было чрезвычайно важно т.к. отсутствовал официальный стандарта. В 1989 году состоялся выход C++ версии 2.0. 

2.Где используется С++.

На языке программирования C++были созданы такие приложения и сайты как: Counter-Strike, браузеры Mozilla Firefox и Internet Explorer, Unity, Unreal Engine и ОС, а также ОП Windows.

3.Почему С++.

Производительность. Благодаря своей вычислительной мощности язык обеспечивает высокую скорость исполнения кода. Отсутствие существенной нагрузки. Язык не утяжеляет программы, позволяет использовать их даже на старых устройствах. Универсальность. Возможно, этот главный плюс C++. Также плюсы находятся в нзвании языка.

Paskal

1.История Paskal.

Язык Паскаль был разработан в 1970 г. Никлаусом Виртом как язык, обеспечивающий строгую типизацию и интуитивно понятный синтаксис. Он был назван в честь французского математика, физика и философа Блеза Паскаля. Одной из целей создания языка Паскаль Никлаус Вирт считал обучение студентов структурному программированию.

2.Где используется Paskal.

Язык программирования Паскаль широко применяется в области обучения программированию. Он предоставляет студентам простой и понятный синтаксис, что делает его идеальным для изучения основ программирования. Кроме того, Паскаль может использоваться для создания простых программ и решения задач в различных областях.

3.Почему Paskal.

Pascal — предельно простой и понятный язык. В нём нет сложных конструкций, перегрузок операторов или неявного приведения типов. Достаточно выучить 15 команд, и вы уже сможете решить большинство школьных задач. Программы на Pascal легко читать и понимать, даже если вы видите этот язык первый раз.
